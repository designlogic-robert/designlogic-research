# USS Research Papers – Repository Structure

This repository contains the research briefs, specifications, and technical papers 
that define the USS ecosystem (UST, USR, USE, CE).

## UST (Universal Semantic Tokens)
- semantic-tokens/: Core ST specification and research briefs
- teleo-tokens/: TeleoToken spec, Teleogenic Layer work
- trade-tokens/: TradeToken spec (FinCE domain)
- control-protocols/: SCP, TCP, TrCP briefs

## USR (Universal Semantic Runtime)
- Runtime architecture papers
- Routing, invariants, validation, planning, runtime type system

## USE (Universal Semantic Engine)
- Engine architecture
- Domain engine interfaces
- Deterministic–probabilistic hybrid papers

## CE (Cognition Engine)
- Cognitive processing layers
- Operator profiles
- Postures, CAP, alignment theory derivatives
