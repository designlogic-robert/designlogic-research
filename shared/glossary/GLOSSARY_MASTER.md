# USS Master Glossary (Working Draft)

This glossary contains canonical definitions and terminology shared across UST, USR, USE, and CE.

> Upon upgrading to ChatGPT Pro, this glossary will be migrated into the ChatGPT Library for cross-paper consistency.

## Terms
- **UST** – Universal Semantic Token. A typed semantic unit with invariants.
- **USR** – Universal Semantic Runtime. Deterministic meaning-processing layer.
- **USE** – Universal Semantic Engine. Execution engine that interprets UST structures.
- **CE** – Cognition Engine. Cognitive reasoning posture layer.
- **Invariant** – A rule or constraint that must always hold true for a token or runtime plan.
- **Semantic Core** – The shared semantic substrate across USS components.
