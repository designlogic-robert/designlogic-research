UST provides a universal, stable representation layer.  
All Cognitive Engines (SynCE, QLE, FinCE) and all semantic engines depend on UST for structured meaning.

UST ensures:
- deterministic meaning formation  
- cross-domain compatibility  
- machine-interpretable semantic features  
- consistent invariants
